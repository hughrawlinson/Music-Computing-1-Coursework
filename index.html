Not so Noisey Music
Algorithmically Generated Music

Overview
Here are two quite distinct projects completed for Tim Blackwell's Music Computing 1 class in 2013. The first takes data from an image made by audiovisual artist Ryoji Ikeda, and sonifies it. The second is a program to algorithmically generate harmonic music based on an iterative algorithm, and trigonometric mapping functions. Please click through to each project to read more about it.
